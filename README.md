# E-Commerce Sales Analysis Project

## Project Overview

This project analyzes a sample e-commerce sales dataset using Python and Pandas. The goal is to explore customer purchasing behavior, sales performance, and revenue trends while finding useful business insights.

## Dataset

The dataset contains 100 sample e-commerce orders with the following columns:

* Order ID
* Order Date
* Customer Name
* City
* Category
* Product
* Membership Type
* Payment Method
* Product Price
* Quantity
* Discount Rate
* Total Amount
* Shipping Duration
* Order Status

## Tools Used

* Python
* Pandas
* Google Colab

## Analysis Steps

1. Imported the CSV file into Google Colab.
2. Explored the dataset using `head()`, `shape`, `info()`, and `describe()`.
3. Calculated summary statistics.
4. Filtered high-value orders.
5. Compared customer spending by membership type.
6. Analyzed revenue by category and city.
7. Created business insights based on the analysis.

## Key Insights

* The average order amount is approximately **1,759 TL**, while the median is **1,173 TL**, showing that some high-value orders increase the average.
* Completed orders generated approximately **158,272 TL** in total revenue.
* The **Electronics** category generated the highest revenue.
* Gold members spent more on average than Standard members.
* Order amounts vary significantly, suggesting that customer segmentation could help improve marketing strategies.

## Business Recommendations

* Focus on retaining high-value customers.
* Invest more in the best-performing product categories.
* Create targeted campaigns for Gold and Silver members.
* Monitor high-value orders and improve the shipping experience.
