E-Commerce Sales Analysis Project
Project Overview
This project analyzes a sample e-commerce sales dataset using Python and Pandas.

The goal of the project is to analyze e-commerce sales trends, evaluate customer segments, examine revenue distribution, and provide data-driven business insights

Dataset
The dataset includes 100 sample e-commerce orders with the following columns:

order id
order date
customer name
city
category
product
membership type
payment method
product price
quantity
discount rate
total amount
shipping duration
order status
Tools Used
Python
Pandas
Google Colab
Analysis Steps
Loaded CSV file into Google Colab
Explored the dataset with head(), shape, info(), and describe()
Calculated summary statistics
Filtered high-value orders
Analyzed customer segments by membership type
Compared revenue by category and city
Generated business insights
Key Insights
The average order amount is higher than the median, which suggests that some high-value orders increase the average.
Completed orders generated a meaningful total revenue.
Some categories contribute more revenue than others.
Gold members tend to have higher average order amounts than standard members.
Order values vary significantly, so customer segmentation can support better marketing decisions.
Business Recommendations
Focus retention campaigns on high-value customers.
Analyze top-performing categories for growth opportunities.
Create targeted campaigns for Gold and Silver members.
Monitor high-value orders and shipping experience closely.
